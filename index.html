Breast Cancer Tumor Classification ‚Äî ML Project

Goal: Developed a supervised machine learning system to accurately classify breast tumors as malignant or benign using real-world diagnostic data.

  • Analyzed a dataset of 569 medical records with 30 diagnostic features (e.g., radius, concavity, symmetry).
  • Performed full data preprocessing: cleaned data, removed noise, scaled features, and applied PCA for visual separation.
  • Trained and compared models: Dummy, Decision Tree, Logistic Regression, KNN, SVM.
  • SVM achieved 97% accuracy (B: Precision 0.99, Recall 0.97 / M: Precision 0.95, Recall 0.97).

Key Skills: ML pipelines, GridSearchCV, data visualization, model evaluation, benchmarking.

Next Steps: Add ensemble models (Random Forest, XGBoost), improve recall with SMOTE.

Technologies: Python, scikit-learn, pandas, NumPy, matplotlib, seaborn
